## 📌 Project Overview
This project focuses on analyzing retail sales and profit performance using Microsoft Power BI. The dataset used is the **Sample Superstore dataset**, which contains information about sales, customers, products, regions, and profits.

The main objective is to convert raw data into meaningful insights using interactive dashboards and visual analytics.

---

## 🎯 Objectives
- To analyze overall sales and profit performance
- To identify high-performing regions and products
- To understand customer segments and their contribution
- To visualize business trends in an interactive way
- To support data-driven decision-making

---

## 📊 Dashboard Overview

The Power BI dashboard includes KPIs, charts, and filters for better analysis.

### 📍 Key KPI Cards
- Total Sales
- Total Profit
- Profit Margin
- Total Orders

---

### 📈 Visualizations
- Sales trend over time (monthly analysis)
- Sales by region
- Profit by category
- Top-performing products
- Customer segment distribution

---

### 🔎 Filters (Slicers)
- Region
- Category
- Order Date

These filters allow users to interact with the dashboard and explore data from different perspectives.
